# 🤖 Discord Bot com MongoDB

Bot desenvolvido em Python utilizando a biblioteca `discord.py` e banco de dados MongoDB.

O projeto possui:
- Sistema de cadastro
- Sistema de moedas
- Daily reward
- Persistência de dados com MongoDB
- Estrutura separada entre bot e banco de dados

---

# 🚀 Tecnologias utilizadas

- Python
- discord.py
- MongoDB
- pymongo
- python-dotenv
